Flujo Completo de Operación
Lista secuencial desde encendido de hardware hasta visualización web incluyendo verificación de conexiones y fixes RTK
1

Encender Base RTK

Asegúrate de que la base Mosaic X5 esté encendida y configurada para transmitir correcciones al Caster NTRIP.

2

Encender Rover

Conecte la alimentación al módulo Ardusimple simpleRTK2B y al 4G NTRIP Master.

3

Verificar Conexión 4G

Observe los LEDs del 4G NTRIP Master para confirmar que ha establecido una conexión exitosa a la red celular.

Verificación
4

Confirmar RTK Fix

Espere a que el rover adquiera un "RTK Fix". Esto suele indicarse mediante un LED específico (verde) en la placa simpleRTK2B.

Verificación
5

Iniciar Servidor Python

Ejecute el script Python que se encarga de recibir los datos NMEA del rover, autenticarlos y enviarlos al backend de Node.js.

6

Iniciar Backend Node.js

Active el servidor Node.js, que gestionará la lógica de la aplicación, el procesamiento de posiciones y la comunicación con la interfaz web.

7

Abrir Interfaz Web

Acceda a la URL de la aplicación web desde un dispositivo con conexión a internet.

8

Seleccionar Modo Real

En la interfaz web, elija la opción "Seguimiento Real" para comenzar a visualizar las posiciones en tiempo real.

Visualización Web

Una vez completados los pasos anteriores, podrá ver el seguimiento en tiempo real de los competidores en la interfaz web.
