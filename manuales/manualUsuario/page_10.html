Flujo del Código Python

1. Conexión TCP

El servidor escucha conexiones entrantes de dispositivos rover

2. Autenticación Base64

Los dispositivos se autentican con un token Base64

3. Recepción Stream NMEA

El servidor lee el flujo de datos NMEA del rover

4. Validación de Fix RTK

Se verifica si la sentencia NMEA tiene indicador "RTK Fix"

5. Parseo y Extracción

Se extraen datos como latitud, longitud y velocidad

6. Envío HTTP POST

Datos procesados se envían en JSON al backend
