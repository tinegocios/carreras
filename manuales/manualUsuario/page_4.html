Componentes de Hardware
Dispositivos RTK incluyendo Mosaic-X5, Ardusimple simpleRTK2B, u-blox ZED-F9P y módulo 4G NTRIP

Base RTK: Mosaic-X5

Estación de referencia fija con coordenadas conocidas con precisión.

448 canales para rastrear múltiples satélites
Soporte para bandas L1, L2 y L5
Tecnología anti-spoofing y mitigación de interferencias

Rover RTK: Ardusimple

Placa base del sistema rover con u-blox ZED-F9P.

Conector XBee para expansión de funcionalidades
Puerto SMA para antena GNSS multibanda
Alimentación vía USB-C o pines dedicados

u-blox ZED-F9P

Receptor GNSS de alta precisión para base y rover.

Soporte para múltiples constelaciones GNSS
Modo de doble frecuencia
Precisión centimétrica
Compatible con protocolo NMEA

Módulo 4G NTRIP

Encargado de la conectividad 4G y recepción de correcciones.

Conexión a Internet a través de red celular 4G
Configuración de parámetros NTRIP (IP, puerto, mountpoint)
Descarga de datos de corrección RTCM
Transmisión de correcciones al rover
Requiere conexión a la red 4G del operador
