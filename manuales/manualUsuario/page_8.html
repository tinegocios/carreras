Comunicación y Red
Implementación de 4G y NTRIP para transmisión de datos GNSS en tiempo real

Red Celular 4G

La red 4G permite conectividad a Internet al rover, permitiendo acceso al servidor NTRIP para recibir correcciones, superando las limitaciones de alcance que presentan los enlaces de radio UHF tradicionales.

Protocolo NTRIP

Estándar basado en HTTP/1.1 para transmisión de datos de corrección GNSS en tiempo real
Modelo cliente-servidor donde el rover actúa como cliente y el NTRIP Caster como servidor

Arquitectura NTRIP

Ventajas del Diseño

Eliminación de la línea de vista entre base y rover, utilizando la infraestructura de Internet
Mayor alcance y cobertura geográfica gracias a la red celular 4G
