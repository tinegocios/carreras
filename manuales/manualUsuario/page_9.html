Servidor Intermedio (Python)
Gateway esencial entre hardware de geolocalización y backend principal

Propósito del Servidor

Servidor Python que actúa como puente entre hardware de geolocalización y backend principal. Recibe datos del rover, los valida, formatea y envía de manera segura para su procesamiento y visualización.

Flujo de Datos

El servidor escucha conexiones TCP entrantes con autenticación Base64

Funcionalidades Clave

Recepción de datos por socket TCP
Autenticación de dispositivos mediante token Base64
Parseo de sentencias NMEA (GGA y RMC)
Envío de datos vía HTTP POST al backend
